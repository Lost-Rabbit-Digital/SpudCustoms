## Capture Tips

### Settings for Best Screenshots
- Resolution: 1920x1080
- Graphics: All effects enabled
- Bloom: Medium (not overwhelming)
- UI Scale: Default

### Timing
- Use Steam's screenshot feature (F12)
- Pause for static shots, unpause for action shots
- Use Story mode for cleaner screenshots (less pressure)

### Post-Processing
- No filters needed - game art style is consistent
- Minor crop adjustments if needed
- Ensure no personal information visible in any UI
